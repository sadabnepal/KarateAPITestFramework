# API Test Framework - Karate

### Getting Started
```
1. git clone https://github.com/sadabnepal/KarateAPITestFramework.git
2. Navigate to KarateAPITestFramework
```

### Run Test
```
mvn test
```

### View Report
```
Path: {PROJECT_ROOT_FOLDER}/target/karate-reports/karate-summary.html
```
